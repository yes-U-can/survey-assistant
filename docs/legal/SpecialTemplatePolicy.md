# Special Template Request Policy (Draft)

## Core Rule
- This middleware is MIT-licensed.
- Source code implementing special templates may be published on GitHub as part of this software ecosystem.

## Consent Requirement (for requesters)
- Before requesting a special template implementation, requester must explicitly agree that:
  1. Implementation source code can be publicly available in this repository or related public repositories.
  2. Public code disclosure does not cancel agreed credit compensation.
  3. Business compensation (credits/payment) and open-source publication are separate dimensions.

## Platform Economic Rule
- Credits/rewards for implementation, listing, and transactions are handled by platform policy.
- Publication of source code does not invalidate settled or pending credit compensation.

## UI Notice Requirement
- The request form must include a mandatory checkbox acknowledging the points above.
- The final submit action must be blocked until consent is checked.
